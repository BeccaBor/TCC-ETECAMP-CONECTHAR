<forte> <h2>Como executar o projeto</h2></forte>
<forte>Siga todos os passos para o site ser executado </forte>

1- Entre no link para baixar o Visual Studio Code: https://code.visualstudio.com/ (Baixa o pacote base para seu sistema e siga os passos de instalação).

2- Entre no link para instalar o xampp: https://www.apachefriends.org/pt_br/download.html (Baixa o pacote base para seu sistema e siga os passos de instalação)

<forte>Instalado os aplicativos vamos para executar o site</forte>

1- Baixe o zip conethar.zip e descompacte ele em alguma área do seu computador

2- Abra ou seu papel de controle do xampp e ativo do MySQL e Apache, clique em admin do MySQL.

3 - Após abr o phpmyadmin em seu navegador, clique em +novo, em nome do banco de dados coloque folhapaga, clique em criar, vá na aba de importar, escolher um arquivo e agora vá ate onde descompactou o zip novo, entre na massa do conectar, e vai estar lá folhapaga, clique nele para importar

4- Abra o Visual Studio Code, no canto superior esquerdo vá até arquivo/file e aperte para abrir pasta/open folder, nova escola a pasta conectar

5- abra o terminal do Vs Code, na barra onde está a massa clique com o botão direito do mouse e vai na operação "abrir terminal integrado"/"Abrir em terminal integrado" vai na operação do cmd

6- Primeiro entre na massa do backend, npm install e depois npm run dev

7- abra outra operação cmd, entre na massa frontend, npm install e por último node.js

8- digite http://localhost:3000

<forte>O site foi iniciado com sucesso!</forte>
